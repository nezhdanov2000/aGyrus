Gyrus AI
Welcome to Gyrus AI! This platform helps students find tutors and manage appointments. This project demonstrates an AI assistant built for a university course using Logistic Regression for intent classification (with TF‑IDF features) in a hybrid UI (text + buttons). You can try: - "Find math tutor" - "Show my bookings" - "Cancel booking"
